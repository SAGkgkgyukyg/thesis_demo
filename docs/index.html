專案 Demo 頁面

這是一個靜態 Demo 頁面，示範如何在瀏覽器端載入 `TFJS` 與 `ONNX` 模型並對 webcam 影像進行即時推論（簡化示範）。

使用流程：先在本機把模型複製到 docs/models/（請看下方指令），然後把變更推到 GitHub，Pages 會部署 /docs。

本機複製指令 
mkdir -p docs/models/tfjs
cp -r models/tfjs/yolov8s_iid docs/models/tfjs/
cp models/*.onnx docs/models/onnx/  # optional: 複製 ONNX 模型
git add docs/models
git commit -m "Add models for GitHub Pages demo"
git push origin main

返回專案首頁（若有啟動伺服器）
